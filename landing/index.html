Autonomous AI Project Management

superbot2

A persistent AI system that manages a portfolio of software projects autonomously. Orchestrator spawns workers. Workers ship code. You review the results.

Install Learn More

Features

Everything you need to run an autonomous software development operation.

□

Spaces

Isolated project areas, each with its own plans, tasks, knowledge, and escalations.

⬡

Orchestrator

Central brain that triages work, spawns space workers, and coordinates across projects.

◈

Dashboard

Real-time web UI with chat, escalation management, scheduling, and system pulse.

⚓

Hooks

PM discipline enforcement. Workers can't go idle without updating tasks, distilling knowledge, and reporting.

≡

Knowledge

Persistent memory across sessions. Conventions, decisions, patterns, and research — nothing gets lost.

✦

Skills

Reusable capabilities: TDD, systematic debugging, brainstorming, implementation pipelines, and more.

⏰

Scheduled Jobs

Cron-like scheduler runs heartbeats, self-improvement analysis, and custom recurring tasks.

↻

Self-Improvement

Analyzes its own conversation logs and suggests improvements to skills, hooks, and workflows.

Architecture

A multi-agent system where the orchestrator delegates work to specialized space workers.
